# Spring Boot 4 Demo 🍃

A sample project showcasing Spring Boot 4 features and capabilities.

## Prerequisites

- Java 21+
- Presenterm

## Quick Start

```bash
# Clone and run
./mvnw spring-boot:run

# Test endpoints
http :8080/hello version==1
http :8080/hello version==2
```

## Presentation

View the Spring Boot 4 presentation:

```bash
presenterm presentation.md
```
